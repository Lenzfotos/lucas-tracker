Google Sheets connection

Connect to your shared Google Sheet so all caregivers see the same data in real time.

How to get your API key:
1. Go to console.cloud.google.com/projectcreate
2. Create a project, then search for Sheets API and enable it
3. Go to APIs & Services → Credentials → Create API Key
4. Restrict the key to Google Sheets API, then paste it below
5. Make sure your sheet is set to "Anyone with link can edit"

About

All entries are appended to your Google Sheet tabs: Food, Sleeping, and BM. Adding new fields in the future will never affect existing data — old rows simply leave new columns blank. Your history is always safe.
